naxoForm 的實際應用：一行對 Form 存取資料、驗證
將資料載入表單 
var form1 = new naxoForm('form1')
form1.Load({
  age: 36,
  isAnonymous: false,
  isStudent: true,
  location: 'Taichung',
  luckyColor: "#563d7c",
  name: 'Peter',
  satisfyScore: 50
})
將表單資料清空 
var form1 = new naxoForm('Form1')
form1.Clear()
取得所有表單資料(json) (console看結果) 
var form1 = new naxoForm('Form1')
form1.GetData()
/* {
  age: 42
  isAnonymous: false
  isStudent: true
  location: null
  luckyColor: "#563d7c"
  name: null
  satisfyScore: 50
  uploadFile: File {name: "photo6055519131...}
} */
取得所有表單資料(FormData) (console看結果) 
var form1 = new naxoForm('Form1')
form1.GetFromData()
// FormData {}
驗證表單資料 
var form1 = new naxoForm('Form1')
form1.Validate()
清除表單驗證 
var form1 = new naxoForm('Form1')
form1.ValidateReset()
